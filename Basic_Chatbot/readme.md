# 🤖 Basic Chatbot

## Overview

A simple rule-based chatbot built with Python. It responds to predefined user inputs such as **"hello"**, **"how are you"**, and **"bye"** using `if-elif` conditions.

## Features

* Responds to basic greetings and questions
* Uses predefined replies
* Runs continuously until the user types **"bye"**
* Beginner-friendly Python project

## Concepts Used

* Functions
* `if-elif-else`
* `while` loop
* User Input/Output
