# 📈 Stock Portfolio Tracker

## Overview

A simple Python project that calculates the total investment value of a selected stock using predefined prices. The user enters a stock name and quantity, and the program displays the total value and saves the portfolio summary to a text file.

## Features

* Displays available stocks
* Accepts stock name and quantity
* Calculates total investment
* Saves the result to `portfolio.txt`
* Handles invalid stock names

## Technologies Used

* Python
* Dictionary
* User Input
* Conditional Statements
* File Handling
